# CampusConnect 🎓

A smart campus event discovery and resource booking platform for university students.

## Features
- 📅 Event discovery and RSVP
- 🏫 Study room and resource booking with conflict detection
- 👥 Study group formation
- 🤖 AI-powered event recommendations
- 🔔 Real-time notifications via WebSockets

## Tech Stack
| Layer | Technology |
|---|---|
| Web Frontend | React + Tailwind CSS |
| Mobile | Flutter |
| Backend API | FastAPI (Python) |
| Database | PostgreSQL |
| Auth | JWT + bcrypt |
| Real-time | WebSockets |
| Deployment | Railway + Vercel |
